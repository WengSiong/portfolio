Internship Project

The website development I am doing is automate network dashboard.

This dashboard is using the html, css, java script to be front-end and python to be back-end. The feature I am doing is using html to do a upload button to let user upload the excel file for raw data and save in the server, after that we will using the python to connect to ssh server. Then, using the for loop to retrieve the raw data one by one and the execute command like “write memory” to cisco device. Then, the result will send back to python and I am using the python to generate the html for show the fully result or filter result to let user more easy to view. For the network dashboard, user can decrease a lot of the process time compare with manually execute command for one-by-one hostname.
